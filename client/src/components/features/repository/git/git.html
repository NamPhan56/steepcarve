# [Resetting a Branch](https://git-scm.com/docs/git-reset) ## Summary This is used to reset a branch to its initial state Nice tutorial: https://devconnected.com/how-to-git-reset-to-head/ ## Commands ``` git reset --hard HEAD~<number-of-commits> git reset --hard HEAD~\ //reset back to branche's HEAD git reset --hard HEAD^ //undo a hard reset git reset --hard HEAD@{1} ``` ## Usage 1. Checkout a branch - ``` git checkout <branch-name> ``` ---
